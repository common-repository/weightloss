== Description ==
Checks for any Woocommerce products missing weights, which may cause postage calculators to malfunction upon customer checkout.

== Installation ==
Upload the Weightloss plugin to your blog, Activate it. Finished!

== Screenshots ==
1. The Weighloss plugin is easily located inside the WooCommerce menu 
2. It will scan all products and show a link to any without a shipping weight
3. Success, once you have updated all missing weights from products!
